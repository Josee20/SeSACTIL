## 주의사항
- 테이블뷰와 마찬가지로 identifier의 설정에 주의하자.
- 이미지뷰에 이미지 넣을 때 자료형 일치
- 라이브러리 사용시 반드시 import하기

## 부족한점
- layout 상수 설정에 대한 이해
- 구조체를 활용하여 자료형을 만들고 데이터를 분산하여 저장하는 것에 대한 이해
- 컬렉션뷰 셀에 함수를 설정해 불러오는 것에 대한 이해
